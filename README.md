# super-sales-marketing-visualization
This Power BI report provides a simple view of sales and marketing performance. It covers key metrics like revenue, profit, customer segments, and regional trends. Interactive visuals help users understand product performance, track sales growth, and find opportunities, making it easier to take better business decisions.
1.Project Title / Headline
Realmart Sales Data Analytics Dashboard

2.Short Description / Purpose
This dashboard analyzes Realmart’s sales performance across cities, product lines, and payment methods to uncover trends and improve business decisions. It helps stakeholders quickly identify top-performing areas and customer behavior patterns.

3.Tech Stack (Power BI) Power BI Desktop – Data visualization & dashboard creation Power Query – Data cleaning & transformation (e.g., removing nulls, formatting columns) DAX (Data Analysis Expressions) – Measures like total sales, average rating, and margins Data Modeling – Relationships between tables (sales, products, cities)

4.Data Source Dataset: Realmart Sales Dataset File: Excel (.xlsx) Link: https://s3.ap-south-1.amazonaws.com/new-assets.ccbp.in/frontend/content/data-analytics/Datasets_DA_Track/Realmart_Sales_Dataset+.xlsx

5.Features / Highlights 📊 KPI Cards (Quick Insights) Example: Total Gross Income = 31.39K, Total Quantity = 11K, Avg Rating = 7.20 → Instantly shows overall performance. 🏙️ City-wise Sales Analysis Example: Mumbai generates 15.4K, higher than Delhi (6.1K) → Helps identify top-performing cities. 💳 Payment Method Insights Example: Cash payments in Mumbai contribute the highest margin (3.3K) → Useful for optimizing payment strategies. 🛍️ Product Line Performance Example: Food & Beverages contributes the highest share (~33%) → Helps in inventory and marketing focus. ⭐ Customer Rating Trends Example: Fashion accessories rated highest (7.29), Home & lifestyle lowest (7.09) → Indicates customer satisfaction by category. 📋 Detailed Transaction Table Example: Invoice-level view showing quantity and income → Enables drill-down analysis. 🔍 Interactive Filters (Slicers) Example: Filter by product line to see only Electronics data → Makes the dashboard dynamic and user-friendly
